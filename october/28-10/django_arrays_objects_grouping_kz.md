
# Практикалық сабақ: Django-да массив және объект әдістері, деректерді топтастыру

## 1. Кіріспе
Бұл сабақта массивтермен (тізімдермен) және объектілермен (сөздіктермен) жұмыс істеудің негізгі әдістерін және деректерді топтастыруды қарастырамыз.

## 2. Массивтермен жұмыс (тізімдер) Django-да
### Негізгі әдістер
- `append()`: элементті тізімнің соңына қосу.
- `extend()`: басқа тізімнің элементтерін қосу.
- `remove()`: бірінші кездескен элементті жою.

### Мысал
```python
products = ["алма", "банан", "алмұрт"]
products.append("апельсин")
print(products)  # ['алма', 'банан', 'алмұрт', 'апельсин']
```

### Тапсырма
Өнімдердің тізімін жасаңыз және оны алфавиттік ретпен сұрыптаңыз.

## 3. Объектілермен жұмыс (сөздіктер) Django-да
### Негізгі әдістер
- `get()`: кілт бойынша мән алу.
- `keys()`: барлық кілттерді қайтару.

### Мысал
```python
user = {"аты": "Алина", "жасы": 30}
print(user.get("аты"))  # Алина
```

### Тапсырма
Пайдаланушы сөздігін жасап, оның жасын жаңартыңыз.

## 4. Django-да деректерді топтастыру
Деректерді топтастыру үшін `itertools` модулінен `groupby` және ORM әдістерін `annotate()` және `values()` қолданамыз.

### `groupby` әдісімен мысал
```python
from itertools import groupby

data = [("жемістер", "алма"), ("көкөністер", "сәбіз")]
data.sort()
grouped_data = {key: list(group) for key, group in groupby(data, lambda x: x[0])}
print(grouped_data)
```

### Тапсырма
Өнімдердің тізімін жасап, оларды санат бойынша топтастырыңыз.
