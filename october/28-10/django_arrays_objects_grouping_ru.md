## 1. Введение
В этом уроке мы рассмотрим базовые методы для работы с массивами (списками) и объектами (словарями) в Django, а также способы группировки данных.

## 2. Работа с массивами (списками) в Django
### Основные методы
- `append()`: добавление элемента в конец списка.
- `extend()`: добавление элементов другого списка.
- `remove()`: удаление первого вхождения элемента.

### Пример
```python
products = ["яблоко", "банан", "груша"]
products.append("апельсин")
print(products)  # ['яблоко', 'банан', 'груша', 'апельсин']
```

## 3. Работа с объектами (словарями) в Django
### Основные методы
- `get()`: получение значения по ключу.
- `keys()`: возвращает все ключи.

### Пример
```python
user = {"name": "Алина", "age": 30}
print(user.get("name"))  # Алина
```

## 4. Группировка данных в Django
Используем `groupby` из `itertools` и методы ORM `annotate()` и `values()` для группировки.

### Пример с `groupby`
```python
from itertools import groupby

data = [("фрукты", "яблоко"), ("овощи", "морковь")]
data.sort()
grouped_data = {key: list(group) for key, group in groupby(data, lambda x: x[0])}
print(grouped_data)
```
