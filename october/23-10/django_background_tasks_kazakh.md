
# Django-дегі фондық тапсырмалар және кезектер

Бұл лекцияда біз Django қолданбасында фондық тапсырмалар мен кезектерді қалай пайдалану керектігін, соның ішінде Celery, Redis, және CRON жұмысын қарастырамыз.

## 1. Фондық тапсырмалардың не үшін қажет?
Фондық тапсырмалар ұзақ уақыт алатын операцияларды орындауға мүмкіндік береді, мысалы, электрондық пошта жіберу, файлдарды өңдеу немесе деректерді экспорттау. Олар қосымшаның негізгі жұмысын бәсеңдетпей, осы тапсырмаларды басқа процестерде орындауға көмектеседі.

## 2. Celery және Redis орнату және баптау
### 2.1 Celery дегеніміз не?
Celery – бұл әртүрлі кезекке қою жүйелерін қолдайтын фондық тапсырмалармен жұмыс істейтін кітапхана. Ол Python тілінде жазылған және оңай бапталады.

### 2.2 Redis дегеніміз не?
Redis – бұл деректерді жадыда сақтайтын ашық көзі бар деректер қоры. Celery тапсырмаларды сақтау үшін Redis-ті пайдалануы мүмкін.

### Орнату:
```bash
pip install celery redis
```

## 3. Celery-мен фондық тапсырмалар құру
### 3.1 Celery қолданбасын баптау
Жаңа `celery.py` файлын жасаңыз және онда келесі баптауларды енгізіңіз:
```python
from celery import Celery

app = Celery('my_project', broker='redis://localhost:6379/0')
app.config_from_object('django.conf:settings', namespace='CELERY')
app.autodiscover_tasks()
```

## 4. Периодтық тапсырмалар және CRON
### 4.1 Celery Beat дегеніміз не?
Celery Beat – бұл периодтық тапсырмаларды баптауға мүмкіндік беретін жоспарлау жүйесі. Мысалы, белгілі бір уақытта жұмыс істеуі керек тапсырмаларды орнатуға болады.

### Мысал:
```python
from celery import shared_task

@shared_task
def send_email_report():
    # Электрондық поштаны жіберу коды
```

## 5. Мысалдар мен толығырақ баптаулар
Тапсырмалар және олардың нәтижелері туралы толық мысалдар лекцияда ұсынылады.
