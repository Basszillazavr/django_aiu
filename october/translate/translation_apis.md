
# Бесплатные API для переводчиков

## 1. LibreTranslate
- **Описание**: Открытый и бесплатный API для перевода, который поддерживает множество языков.
- **Особенности**:
  - Полностью открытый код.
  - Возможность развернуть собственный сервер.
- **Ограничения**: Бесплатный доступ ограничен на публичных серверах.
- **Документация**: [LibreTranslate](https://libretranslate.com/)

## 2. Yandex Translate
- **Описание**: Бесплатный API для перевода текста, поддерживающий множество языков.
- **Особенности**:
  - Поддержка множества языков и интеграция с различными приложениями.
  - Простой и удобный API.
- **Ограничения**: Ограничение по числу запросов в бесплатном тарифе.
- **Документация**: [Yandex Translate API](https://cloud.yandex.com/en-ru/services/translate)

## 3. DeepL Translator
- **Описание**: Высококачественный перевод с поддержкой множества языков.
- **Особенности**:
  - Известен своей точностью и качеством перевода.
  - Простой REST API для интеграции.
- **Ограничения**: Бесплатная версия ограничивает количество запросов и некоторые функции.
- **Документация**: [DeepL API](https://www.deepl.com/pro-api)

## 4. Microsoft Translator Text API (Azure)
- **Описание**: API от Microsoft с поддержкой более 70 языков.
- **Особенности**:
  - Возможности автоматического распознавания языка.
  - Интеграция с другими сервисами Azure.
- **Ограничения**: Бесплатный тариф с ограничением на 2 млн символов в месяц.
- **Документация**: [Microsoft Translator Text API](https://learn.microsoft.com/en-us/azure/cognitive-services/translator/)

## 5. Lingva Translate
- **Описание**: Бесплатный и открытый API, который работает как альтернатива Google Translate.
- **Особенности**:
  - Не требует ключа API.
  - Использует интерфейсы Google Translate.
- **Ограничения**: Публичные серверы могут иметь ограничения по производительности.
- **Документация**: [Lingva Translate GitHub](https://github.com/TheDavidDelta/lingva-translate)

## 6. Google Cloud Translation API
- **Описание**: Один из самых популярных и точных переводческих сервисов.
- **Особенности**:
  - Поддержка более 100 языков.
  - Возможность интеграции в любые приложения.
- **Ограничения**: Бесплатный период с ограниченным числом запросов, после чего доступ к API платный.
- **Документация**: [Google Cloud Translation](https://cloud.google.com/translate)

## 7. MyMemory Translation API
- **Описание**: Один из крупнейших переводческих API, который использует переводческую память.
- **Особенности**:
  - Доступ к огромной базе данных переводов.
  - Возможность добавлять свои переводы в базу.
- **Ограничения**: Бесплатный доступ ограничен по числу запросов.
- **Документация**: [MyMemory API](https://mymemory.translated.net/doc/spec.php)
