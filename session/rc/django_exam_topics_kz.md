# Тақырыптар
- Django веб-фреймворкіне кіріспе
- Django құрылымының негіздері және компоненттері
- URL маршрутизациялау және олардың конфигурациясы
- Django үлгілері (Models) және деректер қорының өзара әрекеттесуі
- ORM (Object-Relational Mapping) негіздері және Django-да оның рөлі
- Django әкімшілік панелі және оны баптау
- Пішіндер мен оларды өңдеу (Forms) Django-де
- Django-да аутентификация және авторизация
- Көрулер (Views) және оларды ұйымдастыру әдістері
- Шаблондар жүйесі (Templates) және статикалық файлдармен жұмыс
- REST API құру Django көмегімен
- Django Signal және оны қолдану
- Django-дағы кэштеу және өнімділікті оңтайландыру әдістері
- Django және сыртқы кітапханаларды біріктіру (Swagger, DRF және т.б.)
- Django-да деректер миграцияларының рөлі
- Django-де деректерді қорғау әдістері (SQL Injection-нен қорғану және т.б.)
- Django-да рольдер және рұқсаттар жүйесін енгізу
- Django және AWS: веб-қосымшаларды орналастыру әдістері
- Жоба құрылымы және көпмодульді қосымшалардағы дұрыс ұйымдастыру әдістері
- Django-да жаһандық күйді басқару (Redux аналогы)
